How to push incremental code:

The following instruction must maintain in the terminal (ctrl+j) after the command promt


  1. git add . (This means add everything)
  2. git commit -m"add a meaningful message"
  3. git push (push git from 'Visual Studio code' html code to Github website)

Then we have to refresh in the github webpage to see changes.
